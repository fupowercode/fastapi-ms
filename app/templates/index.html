Bienvenido a la API de FastAPI Store

Esta API te permite gestionar productos en un almacén. Aquí están las operaciones que puedes realizar:

  • Crear un producto: POST /products/
  • Obtener todos los productos: GET /products/
  • Actualizar un producto: PUT /products/{id}
  • Eliminar un producto: DELETE /products/{id}

Puedes acceder a la documentación interactiva de la API en los siguientes enlaces:

  • Swagger UI - Documentación generada automáticamente.
  • ReDoc - Otra versión de la documentación generada.
